# Desafío Técnico — Ingeniero de QA

## Objetivo

Diseñar e implementar un plan de pruebas automatizadas para un sistema de gestión de clientes.

## Requisitos

- Plan de pruebas funcionales (10+ casos)
- Automatización de 3 casos críticos con Cypress/Playwright
- Reporte de resultados en HTML
- Sugerencia de mejoras al proceso de QA
