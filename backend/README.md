# Desafío Técnico — Desarrollador Backend

## Objetivo

Construir una API REST para la gestión de clientes de Fibex Telecom.

## Requisitos funcionales

- CRUD de clientes (nombre, cédula, dirección, teléfono, plan contratado)
- Endpoint para consultar el historial de pagos por cliente
- Endpoint para activar/suspender un servicio
- Autenticación básica (JWT)

## Stack sugerido

- Node.js + Express / Python + FastAPI / Java + Spring Boot
- Base de datos: PostgreSQL o MySQL
- Documentación: Swagger/OpenAPI
