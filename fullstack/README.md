# Desafío Técnico — Desarrollador Full Stack

## Objetivo

Construir una aplicación completa para la gestión de planes de internet de Fibex Telecom.

## Requisitos

- API REST con Express/FastAPI + base de datos
- Frontend en React/Vue con tabla CRUD y formularios
- Reporte básico de clientes por plan
- Login con roles (admin, agente)
- Todo funcionando con docker-compose
