# Desafío Técnico — Scrum Master

## Objetivo

Analizar y proponer un plan de mejora para un equipo de desarrollo con problemas de entrega en Fibex Telecom.

## Escenario

Un equipo de 7 desarrolladores tiene:
- Velocidad decreciente en los últimos 3 sprints
- Deuda técnica acumulada
- Daily meetings de 45+ minutos
- Bugs recurrentes en producción
- Baja moral del equipo
