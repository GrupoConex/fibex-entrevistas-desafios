## Objetivo

Diseñar e implementar un pipeline CI/CD para la aplicación Fibex Telecom.

- Pipeline CI (lint, tests, build)
- Pipeline CD (deploy a staging/producción)
- Infraestructura como código (Terraform/CDK)
- Monitoreo y alertas básicas
- Docker multi-stage builds
