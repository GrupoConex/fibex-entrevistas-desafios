# Desafío Técnico — Arquitecto de Sistemas

## Objetivo

Diseñar la arquitectura de un sistema de atención al cliente para Fibex Telecom que maneje 10k+ solicitudes concurrentes.

## Requisitos

- Diagrama C4 (Contexto, Contenedores, Componentes)
- Estrategia de escalabilidad y alta disponibilidad
- Decisión tecnológica justificada (base de datos, caché, mensajería)
- Plan de migración desde sistema legacy
- Consideraciones de seguridad y compliance
