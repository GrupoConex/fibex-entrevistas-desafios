# Desafío Técnico — Desarrollador Frontend

## Objetivo

Crear un dashboard interactivo para la gestión de clientes de Fibex Telecom.

## Requisitos funcionales

- Tabla de clientes con filtros (búsqueda, estado del plan)
- Vista de detalle del cliente con historial de pagos
- Formulario para crear/editar clientes
- Gráfico de ingresos mensuales
- Diseño responsive

## Stack sugerido

- React / Vue / Angular
- TypeScript
- CSS Modules / Tailwind / Styled Components
