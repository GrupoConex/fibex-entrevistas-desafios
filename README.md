# Fibex Telecom — Desafíos Técnicos de Entrevistas

Repositorio oficial de desafíos técnicos para el proceso de selección de **Fibex Telecom**.

## Estructura

Cada carpeta contiene el desafío correspondiente a un rol específico:

| Carpeta | Rol | Tiempo estimado |
|---------|-----|-----------------|
| `backend/` | Desarrollador Backend | 4-6 h |
| `frontend/` | Desarrollador Frontend | 3-5 h |
| `fullstack/` | Desarrollador Full Stack | 6-8 h |
| `qa/` | Ingeniero de QA | 3-5 h |
| `arquitecto/` | Arquitecto de Sistemas | 5-7 h |
| `devops/` | Ingeniero DevOps | 4-6 h |
| `scrum-master/` | Scrum Master | 2-3 h |

## Contenido de cada desafío

Cada carpeta contiene:

- **README.md** — Descripción del desafío, objetivos y stack sugerido
- **instrucciones.md** — Pasos detallados para completar el ejercicio
- **criterios.md** — Rúbrica de evaluación con ponderaciones
